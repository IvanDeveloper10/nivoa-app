# Nivoa
Nivoa is an online website that allows you to shop safely and efficiently. It features high-quality products and a well-developed frontend and backend system.
<img width="1898" height="923" alt="Captura de pantalla 2026-04-20 220336" src="https://github.com/user-attachments/assets/b906179b-18ae-4216-90bb-60e36463f68c" />
Nivoa is built with React JS and Firebase for both the frontend and backend. It has a complex system that handles users, authentication, data storage, order management and status, and more. Nivoa is designed for those who need a ready-made e-commerce platform that they can use freely.

Nivoa is an excellent application for entrepreneurs who need to set up an online store and don't have any resources, guides, or templates to use.
